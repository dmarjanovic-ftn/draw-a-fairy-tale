# Draw a Fairy Tale
Bachelor of Computer Science on thesis - Big Data Mining with Scala and Spark

## Description 
[Quick Draw datasets](https://github.com/googlecreativelab/quickdraw-dataset) represents **public** dataset provided by Google. It has various formats of 345 _drawing_ classes. 

## Running
In order to run this project locally, first you need to do is to parse **simplified** `numpy` **datasets** using `python-preprocessing` project. 

Second step is locating to `scala-spark` project and type `sbt spark`.
